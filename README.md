# Consumer-Goods-Ad-hoc-Report
AtliQ Hardwares (imaginary company) is one of the leading computer hardware producers in India and has expanded well in other countries too. Ad-hoc-requests- there are 10 ad hoc requests for which the business needs insights.

# Project Overview

This project focuses on solving a series of ad-hoc business requests using SQL to generate actionable insights from sales, customer, product, and market data. The analysis helps stakeholders make data-driven decisions by uncovering trends, identifying growth opportunities, and evaluating business performance across different dimensions.
